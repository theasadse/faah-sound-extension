# Changelog

All notable changes to the "Faah Sound on Terminal Error" extension will be documented in this file.

## [0.1.1] - 2026-06-13

- Added modern branding logo icon.
- Optimized marketplace metadata description and search keywords.
- Enhanced README documentation with badges and troubleshooting guides.

## [0.1.0] - 2026-06-13

- Initial release of Faah Sound on Terminal Error.
- Plays "faah!" sound when a terminal command exits with a non-zero code.
- Custom sound file configuration.
- Cooldown period configuration to prevent sound spam.
- Cross-platform support for macOS, Windows, and Linux.
